You learned:

reliable communication
ordered packet delivery

Unlike UDP:

You used Linux system calls:

Function	Purpose
socket()	Create socket
bind()	Attach port
listen()	Start listening
accept()	Accept client
close()	Close socket

These are kernel-level APIs.

10. Blocking Calls

Very important concept.

You learned:

accept()

is BLOCKING.

Meaning:

program pauses

This introduces:

11. Linux Process Networking

You observed:

ports exist only while process runs
server lifecycle matters
processes own sockets

When process exits:

socket disappears

This is real operating systems behavior.
